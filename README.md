# Titanfall 2 Multiplayer Fix
Titanfall 2 Intel OpenSSL bug fix

## Instructions
1. Download the "Titanfall2.bat"-file.
2. Edit the file to include your game installation location.
3. Every time you play Titanfall 2, launch the game using the "Titanfall2.bat"-file.
4. If the game launches on windowed mode unintentionally, use alt+enter to change to fullscreen or adjust the window type in the game settings.

You can store the file in any location you want, but easiest would be your desktop as you need to launch the game via the .bat-file every time you play for multiplayer to work. You can also store it in any folder and then create a shortcut to your start menu or desktop if it works better for you.

The next part is completely optional and purely visual. If you create a shortcut it will have the icon of Command Prompt (CMD), but you can change it to the icon of Titanfall 2 by editing the properties of the shortcut. If the shortcut is in the start menu, open the shortcut's file location first by right clicking it and selecting "Open file location". Right click on your shortcut and select "Properties", then click "Change Icon". Look for the icon of Titanfall 2 with the same path as the one you set inside the .bat file for your game installation.
